WEBSITE PORTOFOLIO SEDERHANA

Ditulis oleh Laras Nalendra Sari Pada Hari/tanggal jum'at-20-Juni-2025

Website portofolio sederhana ini merupakan langkah awal saya dalam perjalanan menuju menjadi seorang pengembang web profesional.Saya membuat website ini sebagai bagian dari tugas kuliah, namun juga sebagai sarana untuk menampilkan kemampuan dan minat saya dalam bidang ini.Tujuan utama dari website ini adalah untuk memperkenalkan diri saya dan berita-berita yang inggin saya sampaikan dan informasi tentang data diri saya ,saya ingin kalian bisa melihat dan memperhatihan web saya dengan jelas.

Saya berharap dengan mengerjakan website ini,saya dapat nilai yang bagus sehingga saya lebih semangat dalam perkuliahan ini dan saya dapat mengembangkan kemampuan saya dalam ngoding. Umpan balik dan kritik yang membangun sangat saya harapkan agar website ini dapat terus ditingkatkan dan menjadi portofolio yang lebih representatif di masa mendatang.Nah saya juga mau menjelaskan tentang website saya ini, pertama ada bagian "Beranda,Profil,galeri,Berita,dan Video tentang diri saya" kalo anda penasaran anda bisa menekan tulisan "Beranda,Profil,galeri,Berita,dan Video tentang diri saya" yang ada di atas untuk melihat data diri saya .

Dengan adanya tugas ini membatu saya lebih dalam ,dalam belajar ngoding dan saya percaya bahwa dengan terus belajar dan berlatih, saya akan mampu menciptakan website yang lebih inovatif dan bermanfaat bagi pengguna.
